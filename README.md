Este repositorio tiene la implementación de la prueba técnica solicitada (SQL-.Net-ANGULAR).

La solución incluye:

- Backend desarrollado en .NET 8 utilizando Clean Architecture, Entity Framework y JWT.
- Frontend desarrollado en Angular.
- Implementación de autenticación y autorización por roles.
- CRUD completo de comerciantes.
- Generación de reportes en formato CSV.
- Contenerización mediante Docker (Dockerfile y docker-compose incluidos).
